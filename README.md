# S21_ML6

Данный проект посвящён методам обучения без учителя, а именно — задаче снижения размерности. Он включает в себя сравнительный анализ классических и современных подходов (PCA, SVD, NMF, t-SNE, UMAP, LLE), их применение в задачах визуализации, сжатия изображений и выделения фона из видео, а также в предсказании возраста пользователей по разреженным данным взаимодействий.

<a id="-основные-задачи-проекта"></a>
## 📈 Основные задачи проекта

1) **Снижение размерности и визуализация**

- Применение и сравнение методов: `PCA`, `TruncatedSVD (arpack/randomized)`, `t-SNE`, `UMAP`, `LLE` на датасете `MNIST`.
- Оценка качества вложения с помощью:
  - kNN-классификации (`Macro F1`, `Balanced Accuracy`),
  - `Calinski–Harabasz Index`,
  - Визуального анализа разделения классов.
- Реализация собственного класса автоматизированного сравнителя (comparator) моделей с записью результатов в DataFrame

2) **Работа с разреженными данными**

- Построение разреженной матрицы пользователь–книга из Book Recommendation Dataset.
- Предсказание возраста (`Age`) с помощью:
  - LinearRegression и RandomForest на исходных признаках,
  - тех же моделей на признаках после `PCA` и `UMAP`.

3) **SVD для медиа**

- Сжатие изображений: разложение трёх grayscale-фотографий через `SVD`, восстановление при разных рангах, анализ спектра сингулярных значений.
- Выделение фона из видео: представление видео как матрицы, SVD-разложение, реконструкция фона через low-rank approximation (rank = 1).

4) **Манипуляции с выражением лица**

- Обучение `NMF` на датасете Face Expression Recognition.
- Выделение «улыбочной» компоненты (улыбки) через сравнение активаций (`H`) для классов happy и sad.
- Модификация лиц: добавление/удаление улыбки через коррекцию коэффициентов `NMF`.

<a id="-использованные-данные"></a>
## 📊 Использованные данные

- `Book Recommendation Dataset (Kaggle)`: users.csv, ratings.csv → предсказание возраста;
- `MNIST`: визуализация и сравнение методов снижения размерности;
- `Face Expression Recognition Dataset (Kaggle)`: задача, посвященная выделению улыбки;
- `Video_008.avi (Background Models Challenge)`: выделение фона.

## ⚙️ Установка

```bash

cd S21_ML6
```
2. **Установите зависимости**:
```bash
pip install -r requirements.txt
```
